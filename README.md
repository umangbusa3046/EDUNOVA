# EDUNOVA

> Education Network and Operational Virtual Architecture

EDUNOVA is a modern School ERP (Enterprise Resource Planning) System built using ASP.NET Core Web API, React, SQL Server, and Entity Framework Core.

## Technology Stack

### Backend

- ASP.NET Core Web API
- Entity Framework Core
- LINQ
- JWT Authentication

### Frontend

- React.js
- Bootstrap

### Database

- SQL Server

---

## User Roles

### Admin

- Manage Students
- Manage Teachers
- Manage Accountants
- Manage Classes
- Manage Sections
- Manage Subjects
- Manage Timetables
- Manage Exams
- Publish Results

### Teacher

- View Assigned Classes
- Mark Attendance
- Enter Marks
- View Timetable

### Student

- View Attendance
- View Results
- Download Report Cards
- View Timetable

### Accountant

- Manage Fee Payments
- Generate Receipts
